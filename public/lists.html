Lists (these are hard)

to create an ordered list:

first, use the element < ol >. this will make a numbered list. then, inside the < ol > tag, you can make each individual point on the list with the element < li >.

example of code:

< ol>

< li> point one < /li >

< /ol >

what the code would look like:

  1. point 1

to create an unordered list:

first, use the element < ul >. this will make a unnumbered list. then, inside the < ul > tag, you can make each individual point on the list with the element < li >.

example of code:

< ul>

< li> point one < /li >

< /ul >

what the code would look like:

  • point 1
